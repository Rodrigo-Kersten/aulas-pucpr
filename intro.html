NOÇÕES BÁSICAS DO AMBIENTE R

O “R” é um ambiente de trabalho, não um software ou um APP pronto para uso.

A diferença é que temos que “ensiná-lo” e “prepará-lo” para fazer o que desejamos.

É uma chatice e demanda um monte de aprendizado, é difícil, mas…

A vantagem é que ele não é limitado, pode fazer qualquer coisa que a gente “mandar”; desde cálculos simples, estatística complexa, mapas ou mesmo este site.

Se vocês já instalaram o R podemos começar.

ENSINANDO O R A FUNCIONAR

Instalar pacotes

install.packages("readxl")

este comando faz download das “instruções” para o R aprender a ler arquivos excel. Isso só precisa ser feito uma vez!

É mais ou menos como instalar um APP no celular

PREPARANDO PARA ANÁLISE

Carregar Pacotes já instalados

library(readxl)

com este comando o R carrega as instruções para ler arquivos excel. Isso precisa ser refeito toda vez que o R é aberto.

É mais ou menos como abrir um APP instalado no celular

Definir a pasta (diretório) de trabalho

setwd("C:/Inventario")

este comando avisa o R em que pasta você vai trabalhar. Isso também precisa ser refeito toda vez que o R é aberto.

Carregar dados

Estes comandos carregam teus dados para dentro do ambiente R e já pode trabalhar com eles. Antes sempre verifique o tipo de arquivo.

dados <- read.table("Barigui.txt", header=T)          # texto simples (.txt)

dados <- read.csv("Monica.csv", header= TRUE)         # texto .csv

dados <- read_excel("Restinga.xlsx")                  # excel simples

dados <- read_excel("Palmito.xlsx", sheet = "Seca")   # com várias planilhas

Em todos os casos “dados” passa a ser o nome do conjunto dos teus dados.

Sempre que formos utilizá-los vamos nos referir a ele assim. O R, chama isso de um “objeto”. o conjunto “dados” é um objeto dentro do ambiente

Vocês podem escolher qualquer nome, basta depois trocar também quando forem rodar as análises que faremos

VERIFICANDO/OBSERVANDO OS DADOS

Observar os dados

dados (ou o outro nome que deram ao dados)  # exibe toda sua tabela de dados

head(dados)                                 # exibe as primeiras linhas

str(dados)                                  # exibe os dados por página

tail(dados)                                 # exibe as últimas linhas

str(dados\$PAP)                             # exibe apenas a culuna "PAP"



mean(dados\$PAP)                            # Cálcula a média do "PAP"

sum(dados\$PAP)                             # Soma do "PAP"


class(dados\$PAP)                           # indica que tipo de dado tem na culuna

# Os principais tipos de dados são "numéricos" ou "caracteres", dados   numericos podem ser utilizados em cálculos, dado do tipo caracter é tratado como texto e não entra diretamente em operações matemáticas.

# Caso seja necessário corrigir o tipo de dado:
  
dados\$Ind \<- as.character(dados\$Ind)     # muda o tipo para caractere
dados\$Ind \<- as.numeric(dados\$Ind)       # muda o tipo para numérico

Renomear

O conjunto todo de dados

barigui <- dados

sendo nome.novo <- nome.antigo

seu conjunto de dados agora se chama barigui

Apenas uma coluna (variável)

names(dados)[names(dados) == "Parc"] <- "Parcela"

sendo

  • Parc = nome antigo

  • Parcela = nome novo

Exportando o resultado final para o Excel

Salvar com arquivo do excel

write_xlsx(fito, "result.xlsx")

assim como “dados” são os dados originais “fito” é o objeto (resultado) calculado pelo programa; result.xlsx é o nome do arquivo final em excel, vocês podem trocar por qualquer nome, basta manter .xlsx ao final
